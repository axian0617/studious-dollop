## 被删除的误加页面

| 删除项 | 删除位置 | 原因 |
| --- | --- | --- |
| 左侧全局导航 | `src/components/ui.tsx` | 原交互稿首页不是后台侧边导航结构 |
| 首页“功能入口大面板” | `src/pages/Dashboard.tsx` | 原交互稿首页底部只有 4 个商户服务入口 |
| 首页“绑定身份”入口 | `src/pages/Dashboard.tsx` | 绑定身份属于登录/首次使用流程，不是首页入口 |
| 首页“切换店铺”大按钮入口 | `src/pages/Dashboard.tsx` | 切换店铺应通过顶部商户信息区进入 |
| 首页“信息采集”入口 | `src/pages/Dashboard.tsx` | 信息采集不属于原交互稿首页 4 个入口 |
| “通知”独立全局导航 | `src/components/ui.tsx` | 消息中心管理只作为首页底部入口 |
| “关于/退出”左侧导航 | `src/components/ui.tsx` | 关于和退出只作为首页底部入口 |
| 到店领券页面 | `src/pages/Coupon.tsx` | 该内容来自视觉参考图，不属于本项目业务功能 |
| 领券左侧导航 | `src/components/ui.tsx` | 完整交互总图中没有该高频导航入口 |
| 首页“到店领券”快捷入口 | `src/pages/Dashboard.tsx` | 参考图不可转化为业务入口 |
| `/#/coupon` 路由 | `src/App.tsx` | 删除误加业务页面对应路由 |
| 领券二维码刷新、失效、无券逻辑 | `src/pages/Coupon.tsx` | 删除误加业务逻辑 |
| 后端字段建议中的领券字段 | `docs/handoff.md` | 删除 coupon / voucher / ticket 类字段建议 |

## 功能边界

- 业务功能只以完整交互总图和模块拆解图为准。
- 视觉参考图不生成业务功能。
- 当前项目中不再包含“领券”“优惠券”“到店领券”“券码”“二维码领券”等功能页面或入口。
- 真实业务页面不出现 mock 状态切换按钮；状态验收统一放在 `/#/review`，或通过页面 URL 的 `?state=` query 触发。
