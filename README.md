# 银联商务横屏终端 UI Demo

854×480 横屏硬件终端可交互 Demo，使用 React + TypeScript + Vite + Tailwind CSS。

## 启动

```bash
npm install
npm run dev
```

打开终端输出的本地地址，默认路径为 `/#/login`。

测试账号：

- 手机号：`18889008888`
- 密码：`666888`

## 构建

```bash
npm run build
```

## 目录

- `src/App.tsx`：Hash 路由配置
- `src/components/ui.tsx`：横屏终端通用组件
- `src/pages/`：业务页面
- `src/data/mock.ts`：mock 数据
- `docs/handoff.md`：开发交付文档
